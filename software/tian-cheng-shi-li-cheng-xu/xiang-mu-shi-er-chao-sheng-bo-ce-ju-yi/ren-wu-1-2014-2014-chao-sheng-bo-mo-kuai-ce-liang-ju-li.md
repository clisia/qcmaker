## 4、硬件连接

硬件连接：超声波——表1；液晶显示模块——表2。注意插线时的颜色对应。

![&#x56FE;3.12-5](../../../.gitbook/assets/image355.jpg)

表1

| 超声波 | 甜橙板 |
| :--- | :--- |
| VCC | 5V |
| Trig | 数字口2 |
| Echo | 数字口3 |
| GND | 地 |

表2

| LCD1602显示屏 | 甜橙板 |
| :--- | :--- |
| SDA | 模拟口A4 |
| SCL | 模拟口A5 |
| VCC | 5V |
| GND | GND |

## 5、Q&A

Q：显示屏上没有示数或者示数不变？

A：没有将超声波测量的值赋给变量，首先需要声明变量，赋值变量在初始化程序块外面。
